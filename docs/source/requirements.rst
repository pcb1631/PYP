Requirements checklist
======================

1. Administrator
    - Staff Management: Create, view, update, and delete user accounts for front desk staff and trainers.
    - Classroom Management: View and assign classrooms for fitness classes booked by trainers.
        - This is "venue" in the booking system
2. Member
    - User Profile Management: View personal details and payment history.
        - link to associated functions here
    - Class Booking: Browse, book, and provide feedback for fitness classes.
        - link comment functions here
3. Trainer
    - User Profile Management: View and update personal details.
    - Class Management: Book fitness classes and review feedback provided by members after class completion.
4. Front Desk Staff
    - Member Management: Register new members, renew memberships, and record member check-in/check-out.
        - Check-in / check-out is simply attendance in this project
    - Payment Management: Handle payments and generate payment receipt.

5. Finance Manager
    - Expense and Revenue Management: Record expenses, track revenue, and monitor overall financial transactions.
    - Report Generation: Generate financial and performance reports for management review.
